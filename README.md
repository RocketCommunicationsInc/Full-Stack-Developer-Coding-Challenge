# Full-Stack-Developer-Coding-Challenge

## The Project

## Backend
Python - FastAPI, SQLAlchemy

**Set Up**
```
cd ./backend
pip3 install -r requirements.txt
```

**Run Backend**
```
uvicorn main:app --reload
```

**Load/Reset Test Data**
```
GET - http://127.0.0.1:8000/loadtestdata
```

**Docs**
```
http://127.0.0.1:8000/docs
```
